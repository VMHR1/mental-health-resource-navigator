## Auto-Increment Rules
When modifying a file, increment its version number in ALL HTML files that reference it:
1. If modifying `app.js` → increment `app.js?v=X` in `index.html`
2. If modifying `styles.css` → increment `styles.css?v=X` in `index.html`, `program.html`, `submit.html`
3. If modifying any JS file in `js/` → increment in `index.html` (and `program.html` if used there)
